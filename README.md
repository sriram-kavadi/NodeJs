# Node.js Lab Experiments 
This repository contains the implementation of the first 7 experiments from the **Node.js Lab** (ACE Engineering College, II B.Tech II Semester).
---
## 📌 Technologies Used
* HTML5, CSS3
* JavaScript (ES6)
* Bootstrap
* Java (JDBC, Servlets)
* XML (DTD & XSD)
* MySQL / Oracle DB
---
## 📂 List of Experiments
### 🔹 Experiment 1: Responsive Shopping Cart Web Application
**Description:**
Developed a multi-page shopping cart web application with:
* Home, Login, Registration, Catalog, Cart pages
* Responsive layout using **Flexbox & CSS Grid**
**Key Concepts:**
* HTML structure
* CSS styling
* Responsive design (Media Queries)
---
### 🔹 Experiment 2: Bootstrap Enhancement
**Description:**
Enhanced the shopping cart application using **Bootstrap 5**.
**Features:**
* Responsive Navbar
* Bootstrap Grid System
* Cards for product display
* Styled Forms and Tables
**Outcome:**
Improved UI and mobile responsiveness.
---
### 🔹 Experiment 3: Client-Side Validation using JavaScript
**Description:**
Implemented validation for:
* Registration form
* Login form
* Cart inputs
**Key Concepts:**
* JavaScript DOM
* Form validation
* Input constraints
---
### 🔹 Experiment 4: ES6 Features & Weather App
**Description:**
Built a weather application using **OpenWeatherMap API**.
**Concepts Used:**
* Arrow functions
* Promises
* Async/Await
* API fetching
* Data visualization (charts)
---
### 🔹 Experiment 5: JDBC CRUD Application
**Description:**
Developed a Java application to connect with a database and perform:
* Create
* Read
* Update
* Delete operations
**Technologies:**
* Java
* JDBC
* MySQL / Oracle
---
### 🔹 Experiment 6: XML Validation using DTD & XSD
**Description:**
Created an XML document for a bookstore and validated it using:
* DTD (Document Type Definition)
* XSD (XML Schema Definition)
**Key Learning:**
* XML structure
* Data validation techniques
---
### 🔹 Experiment 7: Servlet Controller Integration
**Description:**
Designed a **Java Servlet Controller** to connect:
* Frontend (Shopping Cart App)
* Backend (Database)
**Concepts Used:**
* Servlets
* Request & Response handling
* MVC architecture basics
---
## 🚀 How to Run
### For Frontend Experiments (1–4)
1. Open project folder
2. Run `index.html` in browser
### For Java Experiments (5 & 7)
1. Install JDK & configure environment
2. Use IDE (Eclipse / IntelliJ)
3. Setup database connection
4. Run Java files
### For XML Experiment (6)
1. Open XML file
2. Validate using DTD/XSD tools or IDE
